Why you don't have to trust us

Kaspa's Toccata upgrade added covenants: spending rules enforced by every node on the network. This raffle is built entirely out of them.

No keys, no theft
There is no wallet holding the pot and no admin key. The covenant is the custodian — the 50/40/10 split is the only spend the network will accept.
Chain-picked winner
The winner comes from the hash of a real Kaspa block, weighted by entry size. Biasing it would require burning real proof-of-work.
Anyone can draw
Settlement is permissionless. If our server vanished forever, any user could trigger the draw and the winner still gets paid.
Reclaim guarantee
If no draw happens within 24h of close, every entrant can take their own coins back. Funds can never be stranded — by design, not by promise.

Don't trust — verify

Every entry address is derived from the public covenant contract. Check it yourself.
